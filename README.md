# Week 1 – Data Preprocessing & Exploratory Data Analysis (EDA)

## AnalystLab Africa Machine Learning Internship

### Project

Titanic Dataset – Data Preprocessing & Exploratory Data Analysis

### Objective

The objective of this project is to clean, preprocess, and analyze the Titanic dataset to prepare it for future machine learning model development. The target variable is **Survived**.

### Tasks Completed

* Loaded and inspected the dataset
* Identified and handled missing values
* Checked and removed duplicate records (none found)
* Detected and analyzed outliers using boxplots and the IQR method
* Applied Label Encoding and One-Hot Encoding to categorical variables
* Performed feature scaling using StandardScaler
* Conducted Exploratory Data Analysis (EDA)
* Generated a correlation matrix and heatmap
* Prepared the dataset for machine learning

### Files Included

* `Titanic_EDA_Preprocessing.ipynb` – Complete preprocessing and EDA notebook
* `Titanic_Cleaned.csv` – Cleaned dataset ready for machine learning
* `Tweek1 titanic report.pdf` – Project report with explanations, visualizations, and insights


### Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

### Key Insights

* Female passengers had higher survival rates than male passengers.
* First-class passengers were more likely to survive.
* Ticket fare showed the greatest variation and was associated with survival.
